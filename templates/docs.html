What (Apa)?

Website ini adalah sistem manajemen HMSI yang mencakup berita, dokumentasi, publikasi, dan jadwal kegiatan.

Why (Mengapa)?

Dibuat untuk mempermudah pengelolaan informasi dan komunikasi dalam organisasi mahasiswa.

Who (Siapa)?

Dikelola oleh pengurus HMSI dan digunakan oleh mahasiswa serta dosen.

When (Kapan)?

Website ini dikembangkan sejak tahun 2025 dan akan terus diperbarui.

Where (Di mana)?

Dapat diakses secara online melalui domain yang telah terdaftar.

How (Bagaimana)?

Dikembangkan dengan Flask untuk backend, Tailwind CSS untuk frontend, serta database MySQL untuk penyimpanan data.

Penjelasan Struktur Source Code

        /hmsi_project
        ├── /static          # Folder untuk aset statis (CSS, JS, gambar)
        ├── /templates       # Folder untuk template HTML
        │   ├── about.html       # Halaman tentang HMSI
        │   ├── base.html        # Template dasar untuk halaman lain
        │   ├── berita.html      # Halaman berita
        │   ├── docs.html        # Halaman dokumentasi website
        │   ├── dokumentasi.html # Halaman dokumentasi teknis
        │   ├── index.html       # Halaman utama (beranda)
        │   ├── kegiatan.html    # Halaman daftar kegiatan
        │   ├── profile.html     # Halaman profil pengguna atau HMSI
        │   ├── publikasi.html   # Halaman publikasi konten
        │   ├── visi.html        # Halaman visi & misi
        ├── app.py          # File utama untuk menjalankan Flask
        └── requirements.txt # Daftar library yang digunakan

Struktur ini mengikuti pola umum proyek Flask. Folder /templates digunakan untuk menyimpan tampilan HTML, sementara /static digunakan untuk menyimpan file CSS, JavaScript, dan gambar. File app.py adalah titik masuk utama aplikasi.
